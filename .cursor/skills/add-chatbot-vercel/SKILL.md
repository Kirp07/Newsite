# Чат-бот на сайт (Vercel / статика)

Добавление виджета чата на статический сайт или сайт на Vercel без обязательного бэкенда.

## Варианты

1. **Встроенный виджет (клиентский)** — кнопка + панель чата в HTML/CSS/JS. Ответы по ключевым словам или фиксированные; подходит для визитки/портфолио. Не требует API.
2. **Сторонний сервис** — Crisp, Tidio, Tawk, Intercom: вставить один скрипт из документации сервиса в `index.html`.
3. **Бот с бэкендом** — Vercel Serverless (API route) или внешний API: виджет отправляет сообщения на endpoint, ответ приходит с сервера.

## Встроенный виджет (рекомендуется для визитки)

- В конце `<body>` в `index.html` добавить разметку: кнопка-пузырёк + панель (заголовок, список сообщений, поле ввода, кнопка «Отправить»).
- Стили вписать в существующие переменные сайта (`--accent`, `--text`, `--bg-card`) или в отдельный блок для виджета.
- Поведение: по клику на кнопку — открыть/закрыть панель; по отправке сообщения — добавить сообщение пользователя в ленту, вызвать логику бота (ключевые слова или случайный ответ), добавить ответ бота в ленту. Сообщения не сохраняются (или только в `sessionStorage` для сессии).
- Скрипт: один `<script>` в том же файле или отдельный `chat-widget.js`, подключаемый скриптом в визитке.

## Интеграция с генератором визитки

Если визитка собирается генератором из формы: в шаблоне генератора добавить тот же блок виджета (разметка + стили + скрипт), чтобы сгенерированный `index.html` тоже содержал чат.

## Чек-лист

- [ ] Разметка: кнопка + панель с заголовком, лентой сообщений, полем ввода.
- [ ] Стили под дизайн сайта, не ломают мобильную вёрстку.
- [ ] Скрипт: открытие/закрытие панели, отправка, ответ бота (ключевые слова или фиксированные фразы).
- [ ] При использовании генератора — виджет включён в шаблон.
