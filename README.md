# Newsite

Сайт-визитка и двухэтапный генератор (форма → скачивание HTML).

## Безопасность (публичный репозиторий)

- **Секреты не коммитить:** ключи API, пароли, токены, файлы `.env` — в `.gitignore`. Используйте `.env` только локально или переменные окружения на хостинге.
- **Проверка перед push:** `git status` и `git diff` — убедитесь, что в коммите нет файлов с секретами.
- Шаблон для переменных (без значений): `.env.example`.
